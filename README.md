# 执法速检方案SafeAudit-Lex
这是为政府、公共场所安全管理人员**扫描判断**Android平台等设备中媒体、通讯记录、应用数据以实现**辅助执法引导+反恐怖暴力&识别有害特征**的**执法解决方案**
> 「开发者**快速部署**至生产环境」
###### 提供控制台、职员端，以及被扫描人员的检查端
# 规划&介绍
###### SafeAudit-Lex意在降低执法成本的同时让公民对执法知情化，拥有更多的权利保证隐私，从而促进执法的接受度增加
有利于为执法人员树立形象，加强执法管理防止滥用搜查权力

对于**公民智能手机等设备内信息检查**的执行方法现状分析表如下
| 策略 | 人工盘查 | 司法取证 |  SA |
| 便利性 | 立即进行 | 复杂准备 | 授权开始 |
| 复杂度 | 额外培训 | 专业人物 | 普通人 |
| 友好度 | 泄露隐私 | 隐私舆论 | 去标识化 |
| 成功率|低 |高 | 中|
 * SA为**解决方案SafeAudit-Lex**简写
